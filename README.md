# G+4 RCC Building Design using ETABS and SAFE

## Project Overview

This project presents the structural analysis and foundation design of a **G+4 Reinforced Concrete (RCC) residential building**. The superstructure was modeled and analyzed in ETABS, and the column loads were exported to SAFE for foundation design.

This project demonstrates the typical workflow used by structural engineers to analyze buildings and design safe foundations based on structural loads.

---

## Software Used

* ETABS – Structural modeling and analysis of the building
* SAFE – Foundation analysis and design

---

## Building Details

* Structure Type: Reinforced Concrete Frame Structure
* Number of Floors: Ground + 4 Floors (G+4)
* Structural System: Beam–Column Frame
* Foundation Type: Isolated Footings
* Concrete Grade: M20
* Reinforcement: HYSD 500 Rebars

---

## ETABS Modeling and Analysis

The building was modeled in ETABS including beams, columns, slabs, and load cases.

### Loads Considered

* Dead Load
* Live Load
* Load Combinations as per design standards

### Analysis Results

The ETABS analysis generated:

* Column axial forces
* Bending moments
* Shear forces

These column reactions were exported to SAFE for foundation design.

---

## SAFE Foundation Design

The foundation system was designed in SAFE using the column reactions obtained from ETABS.

### Steps Performed

1. Imported ETABS model into SAFE.
2. Verified column loads and load combinations.
3. Modeled isolated footings for columns.
4. Generated mesh for footing slabs.
5. Checked soil pressure distribution.
6. Performed punching shear checks around columns.
7. Designed reinforcement for the footings.

### Results Obtained

* Soil Pressure Contours
* Bending Moment Contours (Mx and My)
* Punching Shear Verification
* Reinforcement Requirement in X and Y directions

---

## Project Workflow

1. Structural modeling in ETABS
2. Structural analysis and load combinations
3. Exporting column reactions to SAFE
4. Foundation modeling and meshing in SAFE
5. Soil pressure and punching shear checks
6. Reinforcement design for isolated footings
